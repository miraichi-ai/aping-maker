# Animake Web

PNG連番画像からAPNG（Animated PNG）およびWebP（静止画）のアニメーションをブラウザ上で生成するウェブアプリケーションです。LINEスタンプの厳しい独自フォーマット・サイズ制限にも対応した検証機能（バリデーション機能）を備えています。

## 主な機能

- **ドロップ＆生成**: 複数のPNG画像を直接ブラウザにドラッグ＆ドロップするだけでアニメーション画像を生成。
- **LINEスタンプバリデータ**: アニメーションスタンプ、メイン画像、エフェクトスタンプといった「LINE Creators Market」向けの専用チェックルールを選択可能。
- **プレビュー機能**: リアルタイムで生成されるアニメーションをブラウザ内で確認（再生・一時停止・クリア）。
- **出力調整**: フレームレート（FPS）、ループ回数、無限ループのON/OFFなどのコントロール。
- **多言語対応**: いつでも日本語（Japanese）と英語（English）をワンクリックで切り替え。
- **すべてローカル**: サーバーに画像をアップロードせず、すべてブラウザ上で処理されるため安全かつ高速です。

## 開発環境と技術スタック

- **Vite**: フロントエンドビルドツール
- **Vanilla JS**: フレームワーク非依存のプレーンなJavaScript
- **[UPNG.js](https://github.com/nichenqin/UPNG.js)**: APNGエンコーディングをブラウザ内で実現するコアライブラリ
- **HTML5 Canvas**: 画像処理とプレビュー描画

## ローカルでの実行方法

1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/miraichi-ai/aping-maker.git
   cd aping-maker
   ```
2. 依存関係をインストールします。
   ```bash
   npm install
   ```
3. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```
4. ブラウザで `http://localhost:5173/` にアクセスします。

## 本番用ビルド

パフォーマンスを最適化した静的ファイルを生成します：

```bash
npm run build
```

`dist` ディレクトリの中に本番稼働用のファイルが生成されます。GitHub Pages, Vercel, Netlify などのプラットフォームでホスティングできます。
